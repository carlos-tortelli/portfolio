Conhecimentos Técnicos e Acadêmicos:

🐍 Python:

Linguagem de programação versátil, com sintaxe simples e legível. Utilizada para automação, análise de dados, desenvolvimento web e inteligência artificial. Tenho domínio de estruturas de controle, funções, manipulação de dados e bibliotecas essenciais.

🌐 HTML:

Linguagem de marcação usada para estruturar conteúdo na web. Crio páginas organizadas com elementos semânticos, acessíveis e bem formatadas, servindo de base para aplicações web responsivas.

🎨 CSS:

Utilizo CSS para estilizar interfaces, aplicar layouts flexíveis e responsivos, além de incorporar animações e transições suaves. Familiaridade com Flexbox, Grid e boas práticas de design.

🐧 Linux:

Conhecimento no uso de sistemas baseados em Linux, incluindo linha de comando, gerenciamento de arquivos e permissões.

🧩 Ubuntu:

Experiência prática com a distribuição Ubuntu, tanto em ambiente gráfico quanto terminal.

🔁 Lógica de Programação:

Capacidade de analisar e resolver problemas de forma estruturada. Conhecimento sólido em estruturas condicionais, laços de repetição, vetores, matrizes e algoritmos recursivos.

🤖 Robótica Computacional:

Experiência em projetos que integram hardware e software. Trabalho com sensores, atuadores e microcontroladores, aplicando conceitos de programação embarcada e automação.

⚙️ Laboratório de Circuitos Digitais:

Conhecimento prático em montagem e simulação de circuitos lógicos. Aplicação de conceitos como portas lógicas, flip-flops e sistemas combinacionais/sequenciais.

💻 Web Lab:

Desenvolvimento de aplicações web utilizando HTML, CSS, JavaScript e frameworks. Participação em projetos completos, do front-end ao back-end, com foco em usabilidade e boas práticas.

📊 Algoritmos e Programação:

Domínio de técnicas de programação e desenvolvimento lógico. Experiência em linguagens como Python, aplicando estruturas de dados e algoritmos clássicos para resolver problemas computacionais.

🌐 Cybercultura:

Estudo crítico das transformações sociais provocadas pelas tecnologias digitais. Entendimento sobre redes sociais, identidade digital, cultura da informação e ética na internet.
